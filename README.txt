== Description ==
Ultimate Sticky Popup & Widgets is a simple, easy and fully-customizable WordPress plugin used to add popup on fixed position like bottom left, bottom right, left side or right side , top left side & top right side with User friendly Settings.

Social Share: 
Shortcode for social share [uspaw_social_share]

= Features =

* You can display any content like contact form, feedback form, HTML or any text content etc
* User can Change Title text, background color, border color,  text color, also user can add icon image too
* Responsive Sticky Popup & Display Any Location (left side ,right side , top left side, top-right side, bottom left side , bottom right side)

== Installation ==
1. Upload `ultimate-sticky-popup-and-widgets` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. You will find Ultimate Sticky Popup & Widgets in your WordPress admin screen.
